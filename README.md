# Ближайшие бары

Скрипт предназначен для анализа данных по барама с сайта data.mos.ru

# Как запустить

Скрипт требует для своей работы установленного интерпретатора Python версии 3.5

Запуск на Linux:

```bash

$ python bars.py путь_к_файлу ваша_долгота ваша_широта# возможно потребуется заменить на python3 в некоторых ОС
# Пример вывода скрипта
Самый большой бар: Спорт бар «Красная машина»
Самый маленький бар: Сушистор
Ближайший, к указанным координатам, бар: Бар «ДЖАНГО»

```

Запуск на Windows происходит аналогично.
